# BabyHeaven API

1. levantar la base de datos
```
docker-compose up -d
```
2. yarn install
```
yarn install
```
3. clonar el .env.template y renombrar la copia a .env
4. cambiar las variables de entorno
5. levantar la base de datos
```
docker-compose up -d
```

6. Ejucutar el SEED
```
http://localhost:3000/api/seed
```

7. levantar: `yarn start:dev`
